# msh
[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg?style=flat)](./LICENSE)

**M**y **SH**ell

## Build
```bash
$ make
```

## Execute
```bash
$ ./msh
```
## Help
```bash
$ ./msh -h
```

## Command Help
```bash
$ ./msh -c help
$ ./msh -c 'help command ...'
```
